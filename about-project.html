👥 ContactHub

A JavaScript-based interactive Contact List Manager.

🎯 Purpose of the System

ContactHub is designed to help users efficiently manage their personal and professional contacts in one place. The system allows users to add, search, edit, and delete contact records directly in the browser without requiring any server or database. It demonstrates core JavaScript DOM manipulation and event-driven programming concepts taught in ITEL 203.

⚡ System Features
✚

Add Contacts

Add new contacts with name, phone, and email through a validated input form.

🔍

Live Search

Instantly filter contacts by name, phone, or email as you type.

✎

Edit Records

Update any contact's information using the inline edit form.

🗑

Delete with Confirm

Remove contacts safely with a confirmation modal to prevent accidents.

✖

Clear Form

Reset all input fields instantly with the clear button.

✅

Form Validation

All fields are validated — name length, phone format, and email format.

🛠 Technologies Used
HTML CSS JavaScript DOM Manipulation Event Handling Form Validation Github VS Code

No external libraries or frameworks were used — pure vanilla HTML, CSS, and JavaScript.

💻 JavaScript Concepts Demonstrated
document.getElementById()

Used to reference all form inputs, buttons, and table elements.

querySelector()

Used to select edit and delete buttons within dynamically created rows.

addEventListener()

Attached to buttons and inputs for click and input events throughout the app.

createElement()

Each table row is dynamically created using createElement("tr").

appendChild()

New rows are appended to the table body after creation.

remove() / filter()

Contacts are removed from the data array and the table is re-rendered.
